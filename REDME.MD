# TREINAMENTO React JS
### Procedimento de criação do projeto

* Comando de criação 
```
npm create vite@latest
```

* Entrar na pasta do projeto
```
npm i 
```

* Instalando a dependencia para estilizar styled components
```
npm i --save styled-components
```
